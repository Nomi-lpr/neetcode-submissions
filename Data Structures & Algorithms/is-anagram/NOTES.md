# is-anagram

## 一句话思路
用长度为 26 的数组统计 s 和 t 每个字母出现次数，逐一相加相减，最后数组全为 0 则说明两个字符串互为 anagram。

## 复杂度分析
- **时间复杂度**：O(n)。  
  第一层 for 循环遍历 s 和 t 的每个字符，次数为 n（n = len(s)）；第二层 for 循环遍历长度为 26 的 count 数组，这步是常数时间 O(26) ≈ O(1)。所以总复杂度由 n 决定，是 O(n)。
- **空间复杂度**：O(1)。  
  因为 count 数组长度为 26，是固定常数，不随着输入变化。
- **能否更优**：  
  这已是最优复杂度。无论哪种方法，至少要遍历一次输入字符串，所以下界就是 O(n)。

## 边界条件清单
- s 和 t 长度不等（如 "a", "ab"）：**已显式处理**（`if len(s)!=len(t): return False`）。
- 空字符串输入（如 "", ""）：**天然覆盖**，空数组全为0，返回 True。
- 含有重复字符（如 "aabb", "bbaa"）：**天然覆盖**，计数会正确累加/抵消。
- 所有字符都一样（如 "aaa", "aaa"）：**天然覆盖**，最后数组全为0。
- 输入有非小写字母：**未覆盖**，本代码只适用于全小写英文字符串；若含其他字符将 index 越界或结果错误。
- 超大输入：**天然覆盖**，不过只要是字符串，不涉及整数溢出。
- 单字符输入（如 "x", "x"）：**天然覆盖**。

## Python 语法笔记
1. `count=[0]*26`  
   - **解释**：创建一个长度为 26、全部元素为 0 的列表（对应 26 个小写字母）。
   - **示例：**
     ```python
     a = [0] * 5  # [0, 0, 0, 0, 0]
     ```

2. `ord(s[i])-ord('a')`  
   - **解释**：将字符 s[i] 映射到 0~25 之间的索引，用于数组计数。例如 'a'→0, 'z'→25。
   - **示例：**
     ```python
     idx = ord('c') - ord('a')  # 2
     ```

3. `for i in range(len(s)):`  
   - **解释**：同时按下标遍历 s 和 t 的每个字符，第 i 位相对应。
   - **常见变体**：也可以直接用 `zip(s, t)` 遍历两个字符串，不过这里需要下标来做计数。

4. `for j in count: if j!=0: return False`
   - **解释**：遍历计数数组，只要有任意一位不是 0，就说明不是 anagram。
   - **示例**：
     ```python
     arr = [0, 1, 0]
     for num in arr:
         if num != 0:
             print("非全零")
     ```

## 面试口述剧本
“最直接的方法是分别统计两个字符串中每个字母的出现次数，然后逐一进行比较，复杂度是 O(n)。如果我们用一个长度为26的计数数组，可以在遍历s时加1，遍历t时减1，最后判断数组是否全为0。这种做法只需一次遍历和一次常数级检查，空间和时间都很优，所以最终复杂度是 O(n) 时间和 O(1) 空间。”
